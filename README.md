## 2.0.0版本预计更新点：

- 升级到express4.x

- 精简前端资源，优化grunt配置

- 前端默认整合`bootstrap`

- 去掉原有默认的`velocity`模版引擎，默认模版引擎改为`jade`

- 对模版引擎接口改造
